lttng-remove-trigger(1)
=======================
:revdate: 5 March 2021


NAME
----
lttng-remove-trigger - Remove LTTng triggers


SYNOPSIS
--------
[verse]
*lttng* ['linkgenoptions:(GENERAL OPTIONS)'] *remove-trigger* [option:--user-id='UID'] 'ID'


DESCRIPTION
-----------
The `lttng remove-trigger` command removes the trigger with the ID 'ID'.

See man:lttng-add-trigger(1) to learn more about LTTng triggers.

Use man:lttng-list-triggers(1) to list the available triggers and their
IDs.

With the `remove-trigger` command, you remove a trigger within the scope
of your Unix user. If your Unix user is `root`, you can remove the
trigger of another user with the option:--user-id option.


OPTIONS
-------
Identification
~~~~~~~~~~~~~~
option:--user-id='UID'::
    Remove the trigger of the Unix user having the user ID 'UID'.
+
You may only use this option if your Unix user is `root`.
